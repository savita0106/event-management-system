# Event Management System (EMS)

## Live Project
https://event-management-system-39mn.onrender.com

## GitHub Repository
https://github.com/savita0106/event-management-system

## Features

### Common
- Login system
- View event details
- Search and filter events

### User
- Event registration
- Seat availability display
- Waitlist enrollment
- My event registrations

### Admin
- Create and edit events
- Manage event attendees
- Event registration analytics

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL (Railway)
- Deployment: Render
